# china-m-notes

本仓库用于归档和发布多个独立 HTML 页面。这些页面内容彼此独立，不针对任何特定域名或具体网站。

## 项目简介

`china-m-notes` 是一个静态 HTML 页面归档仓库。仓库中的每个页面均为独立文件，可直接通过浏览器访问。仓库本身不提供任何服务端功能，仅作为页面文件的存储与分发渠道。

## 目录说明

```
.
├── README.md          # 本说明文件
├── pages/             # 存放所有独立 HTML 页面
│   ├── page1.html
│   ├── page2.html
│   └── ...
└── assets/            # 页面引用的公共资源（样式、图片等，如有）
    ├── style.css
    └── images/
```

- **pages/**：主目录，存放所有独立 HTML 页面文件。每个文件可直接打开。
- **assets/**：可选目录，存放页面共享的静态资源。

## 页面归档说明

- 每个 HTML 页面均为自包含的独立文档，可直接通过浏览器本地打开或部署到任意静态托管服务。
- 页面之间无依赖关系，也不共享后端或数据库。
- 归档内容可能随时间更新、增删，具体以仓库历史为准。

## 维护说明

- 本仓库由维护者不定期更新。
- 欢迎通过 Issue 或 Pull Request 提交建议或修正。
- 请勿在页面中嵌入任何形式的商业推广、诱导点击内容或第三方跟踪脚本。
- 提交新页面时，请确保文件命名清晰，并放置于 `pages/` 目录下。

## 使用方式

1. 克隆或下载本仓库到本地。
2. 用浏览器直接打开 `pages/` 下的 HTML 文件即可查看。
3. 也可将整个仓库部署到任意静态网站托管服务（如 GitHub Pages、Netlify 等）。
